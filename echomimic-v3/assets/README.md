# EchoMimicV3 public test fixtures

Each `demoN_first_frame.png` and `demoN_audio.wav` pair originates from the
public `bytedance/LatentSync` repository:

<https://github.com/bytedance/LatentSync/tree/main/assets>

The PNG files are the first frames extracted from the matching public demo
videos. Use the pairs for local EchoMimicV3 evaluation:

```text
demo1_first_frame.png + demo1_audio.wav
demo2_first_frame.png + demo2_audio.wav
demo3_first_frame.png + demo3_audio.wav
```

These fixtures are for local testing only. Do not redistribute the source
video, audio, faces, or likenesses without independently verifying rights.
